===========
RaceResults
===========

Provides scripts to download and parse race results from several
running web sites.  The assumption is that you are harvesting race
results for a running club, and that you have a CSV file of your
club's membership list.

The scripts are

crrr (http://www.coolrunning.com)
---------------------------------
Requires tidy to be installed.

brrr (http://www.bestrace.com)
------------------------------
Requires tidy to be installed.

crrr (http://www.compuscore.com)
--------------------------------
Requires tidy to be installed.
